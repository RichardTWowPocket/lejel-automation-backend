## CRITICAL RULES (Violations = broken render)

1. ONLY import from "remotion". NEVER import React or any other package.
2. ALL animations MUST use useCurrentFrame() + interpolate() or spring().
3. CSS @keyframes, CSS transitions, and Tailwind animate-* classes are STRICTLY FORBIDDEN.
4. The root element MUST be AbsoluteFill filling the entire frame.
5. CJK text MUST set fontFamily: "'Noto Sans KR', 'Apple SD Gothic Neo', sans-serif".
6. Keep the WHOLE file under ~600 lines. You MUST fully close every JSX tag.
7. The file MUST end with the closing brace of the default export function.
8. NEVER embed literal http(s) URLs. Use staticFile() for local assets only.

## TIMING & SEQUENCING

- Use Sequence from={startFrame} durationInFrames={length} layout="none" for each scene.
- Inside a Sequence, useCurrentFrame() returns LOCAL frame (starts at 0).
- Always use layout="none" on Sequence to avoid unwanted absolute-fill wrapping.
- Scenes are CONTIGUOUS: Scene N ends exactly where Scene N+1 starts. No gaps.

## ANIMATION PATTERNS

### Spring Configs (copy these exact values)
- Smooth reveal: { damping: 200 }
- Natural entrance: { damping: 20, stiffness: 96 }
- Snappy UI: { damping: 20, stiffness: 200 }
- Bouncy: { damping: 8 }

### Enter Animation (applied to every scene)
Use this pattern for scene entrance:
```
const enter = spring({ frame: local, fps, config: { damping: 20, stiffness: 96 } });
const groupOpacity = enter;
const groupScale = 0.93 + enter * 0.07;
const groupY = (1 - enter) * 22;
```

### Exit Animation (starts at voiceEndFrame)
Use this pattern for scene exit:
```
const voiceEndLocal = Math.max(0, voiceEndFrame - sequenceFrom);
const exitStart = Math.min(Math.max(0, voiceEndLocal), Math.max(0, durationInFrames - 2));
const exitLen = Math.min(24, Math.max(10, durationInFrames - exitStart));
const exitProg = interpolate(local, [exitStart, exitStart + exitLen], [0, 1], { extrapolateLeft: 'clamp', extrapolateRight: 'clamp' });
const groupOpacity = enter * (1 - exitProg * 0.98);
const groupScale = 0.93 + enter * 0.09 - exitProg * 0.1;
const groupY = (1 - enter) * 22 + exitProg * 18;
```

### Ambient Pulse (for hero elements)
```
const pulse = interpolate(Math.sin(local * 0.08), [-1, 1], [1, 1.045]);
```

### Staggered Elements
```
const stagger = spring({ frame: local - 4 - index * 5, fps, config: { damping: 18, stiffness: 120 } });
```

### SVG Line Drawing
```
const lineProgress = interpolate(local, [6, 32], [length, 0], { extrapolateLeft: 'clamp', extrapolateRight: 'clamp', easing: Easing.out(Easing.cubic) });
// Use strokeDasharray={length} strokeDashoffset={lineProgress} on SVG line element
```

### Bar Chart Growth
```
const reveal = spring({ frame: local, fps, config: { damping: 20, stiffness: 88 } });
const barHeight = interpolate(reveal, [0, 1], [4, targetHeight]);
```
